    First stable release of SeedStack.

  • SeedStack 15.7 "Gingko" M3

    2015-05-12

  • SeedStack 15.7 "Gingko" M2

    2015-05-12

    First publicly available development milestone of SeedStack 15.7 "Gingko".

  • Creation of SeedStack OSS

    2015-02-01

    Creation of the GitHub organization.
    SeedStack an Open-Source enterprise-class development stack derived from the former PSA Peugeot Citroën internal development stack. This internal version was used for two years by more than 20 projects.

  • First internal release of Seed

    2013-10-14

    After 10 months of development with multiple beta releases, the first stable version of Seed, the Java framework, is released internally, at PSA Peugeot Citroën.

  • First internal release of W20

    2012-09-28

    The first stable version of W20 is released. Since there was no Seed framework at the time, it was used with another, already existing backend framework.
